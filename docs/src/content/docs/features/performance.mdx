## Overview

The **Performance** module helps you set goals, run 360 and manager-led reviews, and collect continuous feedback.

## Main capabilities

- **Goals** — Individual and team goals (OKR-style or custom).
- **Reviews** — 360 and manager-only cycles with configurable templates.
- **Feedback** — Request and give feedback anytime.
- **Reports** — Performance summaries and trends.
