## Overview

HR integrates with tools you already use: job boards, communication, identity, and accounting.

## Available integrations

- **Slack** — Notifications for new applications and approvals.
- **LinkedIn** — Publish jobs and import candidates.
- **Google Workspace** — Calendar and single sign-on.
- **Accounting and ERP** — Export payroll and HR data (contact us for your system).

## Setting up an integration

1. Go to **Settings** -> **Integrations**.
2. Choose the integration and click **Connect**.
3. Follow the OAuth or API key steps in the dialog.
4. Configure which events or data to sync.

For API-based integrations, see the [API Reference](/api/).
