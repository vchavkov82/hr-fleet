3. Create `services/api/migrations/001_initial.sql` with a minimal schema:
```sql
-- Initial schema for HR platform API
CREATE TABLE IF NOT EXISTS employees (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    odoo_id INTEGER UNIQUE,
    name VARCHAR(255) NOT NULL,
    email VARCHAR(255) UNIQUE,
    department VARCHAR(255),
    job_title VARCHAR(255),
    status VARCHAR(50) NOT NULL DEFAULT 'active',
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE INDEX idx_employees_status ON employees(status);
CREATE INDEX idx_employees_odoo_id ON employees(odoo_id);
```

4. Create `services/api/internal/db/queries/employees.sql`:
```sql
-- name: GetEmployee :one
SELECT * FROM employees WHERE id = $1;

-- name: ListEmployees :many
SELECT * FROM employees WHERE status = $1 ORDER BY name LIMIT $2 OFFSET $3;

-- name: CreateEmployee :one
INSERT INTO employees (name, email, department, job_title, odoo_id)
VALUES ($1, $2, $3, $4, $5) RETURNING *;

-- name: UpdateEmployee :one
UPDATE employees SET name = $1, email = $2, department = $3, job_title = $4, updated_at = NOW()
WHERE id = $5 RETURNING *;
```

5. Attempt sqlc generation: `cd services/api && sqlc generate`

   - If sqlc is NOT installed: install via `go install github.com/sqlc-dev/sqlc/cmd/sqlc@latest` then generate
   - Add a `generate` script to services/api/package.json: `"generate": "sqlc generate"`

    return pgxpool.New(ctx, databaseURL)
