3. Create `packages/design-system/tokens/transform.ts`:
   - Export `transformTokens()` function that reads tokens.json
   - Flattens nested structure into CSS custom property names (e.g., `global.color.primary` -> `--color-primary`, `global.spacing.4` -> `--spacing-4`)
   - Resolves token references (e.g., `{spacing.4}` -> the actual value)
   - Outputs light and dark CSS variable maps
   - Can be run as script: `npx tsx packages/design-system/tokens/transform.ts` to regenerate css-vars.ts

4. Create `packages/design-system/tokens/css-vars.ts`:

   - Must include ALL 7 categories as CSS custom properties

   - Export dark mode color palette derived from the light palette
