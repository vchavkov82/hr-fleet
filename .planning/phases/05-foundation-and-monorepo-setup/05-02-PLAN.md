4. Create `packages/design-system/styles/globals.css`:
```css
:root {
  /* Generated from css-vars.ts light theme */
  --color-primary: #1B4DDB;
  --color-primary-light: #3B6EF0;
  /* ... all light mode tokens */
}

@media (prefers-color-scheme: dark) {
  :root {
    /* Generated from css-vars.ts dark theme */
    --color-primary: #3B6EF0;
    /* ... all dark mode tokens */
  }
}

.dark {
  /* Class-based dark mode override */
  --color-primary: #3B6EF0;
  /* ... same as media query dark */
}
```

5. Export globals.css and token types from package.json exports field.
