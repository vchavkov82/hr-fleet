Create app/hr/src/components/calculators/salary-calculator.tsx:
- 'use client' directive
- import { useState, useMemo } from 'react'
- import { BG_TAX_2026 } from '@/lib/bulgarian-tax'
- Props: { labels: Record<string, string> } (translated strings passed from server page)
- State: grossSalary (number, default 2000), mode ('gross-to-net' | 'net-to-gross'), bornAfter1960 (boolean, default true)
- useMemo calculation:
  - Clamp insurable income: Math.min(Math.max(grossSalary, BG_TAX_2026.MIN_INSURABLE_INCOME), BG_TAX_2026.MAX_INSURABLE_INCOME) for social security base
  - Employee deductions: pension + illness/maternity + unemployment + health + universal pension (if born after 1960)
  - Taxable income = gross - employee social
  - Income tax = taxable * 0.10
  - Net = gross - employee social - income tax
  - Employer social: pension + illness/maternity + unemployment + accident + health + universal pension
  - Total employer cost = gross + employer social
  - For net-to-gross mode: iterate (binary search or formula inversion) to find gross that yields target net
- UI: card with two tabs (Gross→Net, Net→Gross)
  - Input: number field with BGN label
  - Toggle: "Born after 1960" checkbox
  - Results section: breakdown table showing each deduction line, total employee, total employer, net salary, total cost
- Use Intl.NumberFormat('bg-BG', { style: 'currency', currency: 'BGN' }) for all monetary display
- Results update in real-time as input changes
