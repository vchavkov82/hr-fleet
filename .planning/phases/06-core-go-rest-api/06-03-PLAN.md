   - `RolePermissions map[Role][]Permission` — hardcoded permission sets
   - `HasPermission(role Role, perm Permission) bool`
   - `RequireRole(roles ...Role) func(http.Handler) http.Handler` — middleware that extracts role from JWT claims and checks
