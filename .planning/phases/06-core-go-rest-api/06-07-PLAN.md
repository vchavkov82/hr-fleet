<task type="auto">
  <name>Task 2: Worker binary, route wiring, and Makefile</name>
  <files>services/api/cmd/server/main.go, services/api/cmd/worker/main.go, services/api/Makefile</files>
  <action>
1. Create `cmd/worker/main.go`:
   - Parse same config as server (DB, Redis, Odoo connection)
   - Create Asynq server with Redis connection
   - Register task handlers: `payroll:process` -> worker.ProcessPayrollTask, `webhook:deliver` -> worker.DeliverWebhookTask
   - Asynq config: concurrency 10, retry 5 for webhooks, retry 1 for payroll
   - Graceful shutdown on SIGTERM

2. Update `cmd/server/main.go` to support `--mode` flag:
   - `api` (default): run HTTP server only
   - `worker`: run Asynq worker only
   - `both`: run both (for development)

3. Wire ALL routes in main.go under `/api/v1`:
   - Global middleware: zerolog logging, OTel tracing, Prometheus metrics, CORS, request ID
   - Public: `POST /auth/login`, `POST /auth/refresh`, `GET /health`, `GET /metrics`
   - Protected (APIKeyOrJWT middleware):
     - Auth: `POST /auth/api-keys`, `GET /auth/api-keys`, `DELETE /auth/api-keys/{id}`
     - Employees: `GET/POST /employees`, `GET/PUT/DELETE /employees/{id}`
     - Contracts: `GET/POST /contracts`, `GET /contracts/{id}`
     - Leave: `GET /leave/allocations`, `GET/POST /leave/requests`, `POST /leave/requests/{id}/approve`, `POST /leave/requests/{id}/reject`
     - Payroll: `GET/POST /payroll-runs`, `GET /payroll-runs/{id}`, `POST /payroll-runs/{id}/approve`, `POST /payroll-runs/{id}/process`
     - Payslips: `GET /payslips`, `GET /payslips/{id}`, `POST /payslips/{id}/confirm`
     - Reports: `GET /reports/payroll-summary`, `GET /reports/tax-liabilities`
     - Webhooks: `GET/POST /webhooks`, `DELETE /webhooks/{id}`, `GET /webhooks/{id}/deliveries`
   - Swagger: `GET /api/docs/*` serving httpSwagger

4. Update Makefile:
   - `make api` — build and run API server
   - `make worker` — build and run worker
   - `make dev` — run both (--mode=both)
   - `make swagger` — regenerate swagger docs
   - `make generate-keys` — generate dev RSA key pair
   - `make sqlc` — regenerate sqlc
   - `make test` — go test ./... -short -count=1
   - `make test-race` — go test ./... -count=1 -race
