  <action>
Create a 'use client' component `FreelancerComparison` that accepts `{ labels: Record<string, string> }` props.

**State (all via useState):**
- monthlyAmount: number (default 3000)
- bornAfter1960: boolean (default true)
- includeIllnessMaternity: boolean (default true)
- includeVat: boolean (default false)
- isAnnual: boolean (default false -- monthly by default)
- vacationDays: number (default 20, min 20)
- overhead fields: accountantFee (294), bankFees (25), adminTime (150), registrationAmortized (40) -- from BG_EOOD_2026.OVERHEAD defaults

**Calculations (all via useMemo):**
- eoodResult = computeEoodNet(monthlyAmount, { bornAfter1960, includeIllnessMaternity, includeVat, overhead })
- employmentResult = computeNetFromGross(monthlyAmount, bornAfter1960)
- dailySalary = monthlyAmount / 21 (working days)
- savings = computeSavings(employmentResult.netSalary, eoodResult.netToOwner, vacationDays, dailySalary)
- employmentEffectiveTaxRate = 1 - (employmentResult.netSalary / employmentResult.totalEmployerCost)
- displayMultiplier = isAnnual ? 12 : 1

**Layout sections (top to bottom):**

1. **Input section** (full width, centered):
   - Monthly invoice amount input (number, min 0, max 50000, step 100) with BGN suffix
   - Born after 1959 checkbox
   - Illness/maternity toggle (default ON, note: "changes rate from 31.3% to 27.8%")
   - VAT registered toggle (default OFF)
   - Monthly/Annual toggle buttons (like existing mode tabs pattern)

2. **Savings banner** (full width, green bg):
   - Green checkmark icon
   - "Save {X} BGN/month" or "Save {X} BGN/year" depending on toggle
   - Show total savings including vacation value
   - Effective tax rate comparison: "EOOD: {X}% vs Employment: {Y}%"

3. **Side-by-side columns** (grid, 2 cols on lg, 1 on mobile):
   - **Left column (EOOD)** -- muted: `bg-gray-50 border border-gray-200 rounded-2xl p-6`
     - Header: "EOOD / Freelancer" with muted icon
     - Revenue line
     - Self-insurance contribution (flat, show base 1077 BGN)
     - Overhead breakdown (each editable input with label):
       - Accountant fee, Bank fees, Admin time, Registration amortized
     - Total overhead subtotal
     - Taxable profit
     - Corporate tax (10%)
     - After-tax profit
     - Dividend tax (10%)
     - **Net to owner** (bold, large)
     - Effective tax rate

   - **Right column (Employment)** -- highlighted: `bg-green-50 border-2 border-green-300 ring-2 ring-green-200 rounded-2xl p-6`
     - "Recommended" badge at top
     - Header: "Employment / Payroll" with green checkmark
     - Gross salary
     - Employee deductions (total, collapsible detail)
     - Income tax (10%)
     - **Net salary** (bold, large, green)
     - Employer cost (shown as info)
     - Effective tax rate

   - When VAT toggle ON, show VAT breakdown at top of EOOD column:
     Invoice excl. VAT + 20% VAT = Total invoice. Note: "VAT is pass-through, doesn't affect net income"

4. **Benefits comparison table** (full width below columns):
   - Two-column checklist: EOOD vs Employment
   - Vacation days input (editable, default 20, min 20) with monetized value shown
   - Rows:
     - Paid annual leave: EOOD X (self-employed, no paid leave) / Employment checkmark ({vacationDays} days, worth {X} BGN)
     - Sick pay (80%): EOOD X / Employment checkmark
     - Maternity (410 days): EOOD warning (limited) / Employment checkmark
     - Unemployment insurance: EOOD X / Employment checkmark
     - Mortgage eligibility: EOOD warning (harder) / Employment checkmark
     - Labor code protection: EOOD X / Employment checkmark

5. **Footnote disclaimer** (small, gray):
   "EOOD may be advantageous if you can negotiate 20-30% higher rates or deduct significant business expenses"

6. **CTA section** (centered, primary-50 bg):
   "Switch to our payroll service -- start free trial" button linking to /auth/sign-up

   - Shows savings amount
