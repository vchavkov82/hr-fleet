## v1.1 — Platform Expansion

*Released: March 2026*

Version 1.1 transforms the HR marketing site into a full platform with a Go REST API backend, admin panel, and comprehensive documentation.

### New Features

- **Go REST API** — Complete backend service with Chi router, PostgreSQL (sqlc), Redis caching, and Asynq background workers. Endpoints cover employees, leave, payroll, contracts, webhooks, API keys, users, and audit logs.
- **Admin Panel** — Vite 6 + React 19 single-page application with TanStack Router. Features employee management, leave tracking, payroll processing, contract management, and system settings.
- **RBAC System** — Seven-role hierarchy with eleven granular permissions. JWT RS256 authentication with API key support for integrations.
- **Odoo 18 Integration** — Custom JSON-RPC client with circuit breaker, two-layer caching (Ristretto L1 + Redis L2), and stale-cache fallback.
- **Payroll Processing** — Asynchronous payroll runs via Asynq with real-time status polling. All monetary values in stotinki (BIGINT) for precision.
- **Webhook System** — HMAC-SHA256 signed event notifications with async delivery queue and configurable scopes.
- **Design System** — Tokens Studio JSON as canonical source for colors, typography, spacing, shadows, borders, breakpoints, and component tokens.
- **Monorepo Migration** — pnpm workspaces + Turborepo with Go services integrated via thin package.json wrappers.
- **Documentation Hub** — This documentation site with architecture guides, API reference, and security documentation.

### Infrastructure

- Podman Compose orchestration with Caddy reverse proxy
- Separate PostgreSQL instances for application (v16) and Odoo (v17)
- GitHub Actions CI/CD with reusable workflow pattern

## v1.0 — Initial Launch

*Released: February 2026*

The initial release of the HR platform focused on the marketing website and core HR tools.

### Features

- Marketing homepage with product hero and social proof sections
- Features page showcasing HR modules with alternating screenshot rows
- Pricing page with module-based subscription model
- HR Tools section with Bulgarian salary calculator and freelancer vs. payroll comparison
- Email-gated document templates for contracts, policies, and HR forms
- Blog integration via Astro subpath proxy
- Free trial sign-up flow without credit card requirement
- AI Assistant powered by Claude for HR policy and calculation queries
- Full Bulgarian and English localization
- Framer Motion scroll animations throughout the marketing site
