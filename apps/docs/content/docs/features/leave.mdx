## Overview

The Leave module handles leave requests, approvals, balances, and team visibility. It supports annual, sick, and custom types with rules that respect Bulgarian labour law. Main capabilities: leave types with accrual rules, request and approval workflow, team calendar, and absence reports. Related: [Core HR](/docs/features/core-hr), [Payroll](/docs/features/payroll).
