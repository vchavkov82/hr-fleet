## Overview

Core HR is the central place for employee data, org structure, and HR documents. It includes employee directory, organization chart, document storage, and role-based access (Admin, HR, Manager, Employee). See [Quickstart](/docs/getting-started/quickstart) to import employees and [Leave](/docs/features/leave) for how leave uses this data.
